This site is based on Prologue, a simple, single page responsive site template by
HTML5 UP. The original template has been substantially modified:


Template modernization:

  - Removed IE/legacy polyfills (html5shiv, CSS3 Pie, background-size polyfill,
    Respond.js)
  - Replaced jQuery, jquery.scrolly, jquery.scrollzer, and Skel.js with vanilla JS
  - Self-hosted web fonts (GDPR compliance; previously loaded from Google Fonts CDN)
  - Upgraded Font Awesome 4 to Font Awesome 6


Visual redesign:

  - Complete dark astronomy-themed palette: deep navy/charcoal backgrounds, gold
    accent colour (#e8a838), off-white body text
  - New typography: Lora serif headings (self-hosted), Source Sans Pro body
  - Hero section: circular photo with gold ring, layered over starry background
    image with darkened gradient overlay
  - Research section: responsive CSS grid of cards with banner images, icon badges,
    and scroll-reveal animations
  - Publications section: key-metric tiles (papers / first-author / citations),
    curated first-author highlights list, portal buttons to ADS and ORCID
  - Codes section: horizontal repository rows with logo area, instrument and
    peer-review badges, install snippets, and paper links
  - Floating back-to-top button (replaces per-section back-to-top links)
  - Removed overlay.png noise texture throughout


New pages:

  - cv.html: standalone HTML CV page with sticky navigation bar, print stylesheet,
    and PDF download fallback; uses the same CSS as the main site


Accessibility:

  - Semantic landmark elements: <header>, <main>, <footer>, <nav> replace the
    original <div id="header">, <div id="main">, <div id="footer"> wrappers
  - Skip-to-content link on both index.html and cv.html
  - alt text on all images
  - aria-label on ambiguous links ("Details", "Download")
  - aria-hidden="true" on all purely decorative Font Awesome icons
  - aria-current="true" set dynamically by the scrollspy on the active nav link
  - WCAG AA contrast ratios verified for all text/background combinations
  - prefers-reduced-motion respected for card animations
  - focus-visible outline styles for keyboard navigation
  - Minimum 44×44 px touch targets on interactive elements


Dependencies:

  - Font Awesome 6.5.1 (CDN, SRI hash pinned)
  - Bootstrap 5.3.3 (CDN, SRI hash pinned) — used only on
    Exo3Conference_poster.html (upgraded from Bootstrap 3.4.1)
  - No other external JavaScript or CSS dependencies
